Personal Website

I’ve always wanted a personal website and now I finally have one.

This site is built from the ground up with plain HTML, CSS, and JavaScript. It’s fully responsive and automatically adapts to your system’s light or dark mode preferences.

Updates are fully automated: GitHub Actions build the site into a Docker image and publishes it to the GitHub Container Registry (GHCR). The image is then deployed to a VPS, where it runs behind a reverse proxy to ensure fast and secure access.
